    General Syntax¶

    MarkDown Cheat Sheet

    1. Headings # h1 Heading ## h2 Heading ### h3 Heading #### h4 Heading ##### h5 Heading ###### h6 Heading
    2. Horizontal Rule ___: three consecutive underscores ---: three consecutive dashes ***: three consecutive asterisks
    3. Emphasis
      • Bold **rendered as bold text**
      • Italics _rendered as italicized text_
      • BlockQuote > Blockquotes
      • Nested BlockQuote > First Level >> Second Level >>> Third Level

    4. List

      • Unordered * or + or -
      • Ordered 1.

    5. Code Block ` or [```html] (3 backticks and follwed by the language)

    6. Links

      • Inline Links [Text](http://text.io)
      • Link titles [Text](https://github.com/site/ "Visit Site!")

    7. Images ![Minion](http://octodex.github.com/images/minion.png)
